## Project-Specific Development Rules

- For any frontend UI or UX design task in this project, AI assistants must use the skill at `.agents/skills/ui-ux-pro-max/SKILL.md` before implementing or revising the interface.
- This requirement applies to UI creation, redesign, visual polish, layout changes, interaction refinement, and UI-focused review work.
- The skill output should inform the design system, visual direction, typography, color usage, interaction patterns, responsiveness, and accessibility decisions.
